# 股权投资 - private-equity

面向PE/VC投资团队的股权投资全流程工具。覆盖项目筛选、尽调清单、条款审查、投决备忘录、回报测算到退出分析，支撑从Deal Sourcing到退出的完整投资闭环。

## 子技能列表

| 子技能 | 功能 | 触发关键词 |
|--------|------|-----------|
| 审条款 | TS/SPA文件审查，逐项评估风险等级并给出谈判建议 | 条款审查, 审TS, 审SPA, term sheet |
| 尽调清单 | 项目描述生成结构化尽调清单 | 尽调清单, due diligence, DD checklist |
| 投决备忘录 | 项目信息+尽调发现输出投委会IC Memo | 投决备忘录, IC memo, 投委会 |
| 测收益 | 交易条款+退出假设输出IRR/MOIC/DPI测算 | 回报测算, 算IRR, 回报率, MOIC |
| 筛项目 | BP/CIM生成一页项目筛选备忘录 | 项目筛选, deal sourcing, 项目初筛 |
| 退出分析 | 被投公司现状输出退出路径对比报告 | 退出分析, 退出路径, IPO退出 |

## 使用方法

通过 Marvis 对话自然触发，说出需求即可自动匹配对应子技能。

## 协同技能

- 投研分析：尽调前先做行业研究和可比公司分析
- 企业法务：条款审查和尽调清单需配合法务风险评估
- 企业财税：回报测算需配合财税做税务影响分析
