# 筛项目 -- PE/VC项目快速筛选

> 本技能无需连接器即可完整使用——以上功能会通过手动输入来替代。

快速筛选评估PE/VC项目，从商业计划书（BP）或保密信息备忘录（CIM）中提取关键信息，按投资逻辑框架输出一页筛选备忘录。

**重要提示**：本技能辅助投资团队快速筛选项目，不构成投资建议。项目筛选涉及商业判断，建议投资经理结合实际情况确认。

## 输入要求

确认：
1. **项目材料**：BP/CIM/项目描述
2. **基金信息**（如有）：人民币基金/美元基金、阶段偏好、行业偏好、单笔投资区间
3. **关注维度**（如有特别要求）

**输入不完整时**：
- 只有项目名称/行业 --> 基于公开信息初步评估，标注"[信息有限，建议补充BP]"
- 有BP无基金信息 --> 按通用标准评估
- 非标材料（新闻/融资公告）--> 提取可得信息，标注信息缺口

## 执行流程

### 第零步：红线快筛（一票否决）

| 红线 | 判断标准 | 处置 |
|------|---------|------|
| 行业政策禁区 | P2P/ICO/课外学科培训/开采类无证矿企 | 直接放弃 |
| 创始人重大负面 | 失信被执行人/刑事案件/行业黑名单/多次创业均失败且无反思 | 直接放弃 |
| 财务造假嫌疑 | 收入/利润与现金流严重背离（经营现金流/净利润<0.3连续2年）、审计非标 | 直接放弃 |
| 核心资质缺失 | 须持牌经营但无牌照且12个月内无法获取 | 直接放弃 |
| 估值严重脱离 | 估值倍数>同行业同阶段中位数3倍以上且无合理解释 | 标记高风险 |

如触发红线，直接输出简要结论，无需继续后续步骤。

### 第一步：关键信息提取

从BP/CIM中提取：

| 维度 | 提取内容 |
|------|---------|
| 基本面 | 公司名称、成立时间、注册地（境内/VIE/红筹）、行业、阶段 |
| 业务模式 | 做什么、卖给谁、怎么赚钱、核心价值主张 |
| 团队 | 创始人背景、核心团队完整性、稳定性 |
| 财务 | 收入规模、增速、毛利率、净利率、burn rate |
| 融资历史 | 历史轮次/金额/估值/投资人 |
| 本轮融资 | 金额、估值(pre/post)、资金用途 |

### 第二步：六维量化评分

每维度1-5分，打分锚点如下：

**赛道评估**（TAM/增速/政策/竞争格局）：

| 分值 | 锚点 |
|------|------|
| 5 | TAM>500亿且CAGR>30%，政策明确鼓励，竞争格局未定 |
| 4 | TAM>200亿且CAGR>20%，政策中性偏正，有明确空间 |
| 3 | TAM>100亿且CAGR>15%，竞争激烈但有差异化空间 |
| 2 | TAM<100亿或CAGR<10%，红海竞争 |
| 1 | 市场萎缩/政策限制/行业天花板明显 |

**商业模式**（盈利清晰度/单位经济/可复制性）：

| 分值 | 锚点 |
|------|------|
| 5 | LTV/CAC>5，毛利>70%，强网络效应/规模效应 |
| 4 | LTV/CAC>3，毛利>50%，模式可复制性强 |
| 3 | LTV/CAC>2，毛利>30%，单位经济模型成立 |
| 2 | LTV/CAC<2或毛利<30%，盈利路径不清晰 |
| 1 | 商业模式未验证/持续烧钱无拐点 |

**竞争壁垒**（技术/品牌/渠道/数据/规模/牌照壁垒）：

| 分值 | 锚点 |
|------|------|
| 5 | 核心专利/独占资源，竞争者复制需>3年 |
| 4 | 明确壁垒（数据积累/品牌认知），复制需1-3年 |
| 3 | 有一定壁垒但非不可逾越，先发优势 |
| 2 | 壁垒薄弱，竞争者6-12个月可追上 |
| 1 | 无明显壁垒，完全可被替代 |

**团队评估**（行业认知/执行力/融资能力/互补性）：

| 分值 | 锚点 |
|------|------|
| 5 | 连续成功创业者/行业顶尖专家，核心岗位齐备，里程碑超额完成 |
| 4 | 行业>5年+成功创业经历，团队互补，执行力强 |
| 3 | 相关行业背景，团队基本齐备，执行中规中矩 |
| 2 | 行业经验不足或团队不完整（缺CTO/缺销售负责人） |
| 1 | 跨行业创业且无相关资源，团队不稳定/核心离职 |

**财务健康度**（增速/毛利趋势/现金安全边际）：

| 分值 | 锚点 |
|------|------|
| 5 | 收入增速>100%（早期）或>50%（成长期），现金跑道>18个月 |
| 4 | 增速>50%（早期）或>30%（成长期），跑道>12个月 |
| 3 | 增速>30%（早期）或>20%（成长期），跑道>9个月 |
| 2 | 增速<20%或跑道<6个月 |
| 1 | 负增长或现金将耗尽（<3个月） |

**估值合理性**（PS/PE倍数 vs 可比）：

| 分值 | 锚点 |
|------|------|
| 5 | 估值<同行业同阶段中位数0.7x |
| 4 | 估值在中位数0.7x-1.0x |
| 3 | 估值在中位数1.0x-1.5x |
| 2 | 估值在中位数1.5x-2.5x |
| 1 | 估值>中位数2.5x且无合理溢价解释 |

### 阶段差异化评估

| 阶段 | 权重侧重 | 可弱化 | 核心判断 |
|------|---------|--------|---------|
| 天使/种子 | 团队x2、赛道x1.5 | 财务、估值 | "对的人做对的事" |
| A轮 | 模式x1.5、壁垒x1.5 | 盈利性 | PMF信号是否存在 |
| B轮 | 财务x1.5、模式x1.5 | — | 增速能否证明可复制 |
| C轮/Pre-IPO | 财务x2、估值x1.5 | 团队（已验证） | 利润率和IPO可行性 |

### 第三步：风险矩阵

按影响程度(高/中/低) x 发生概率(高/中/低)绘制风险矩阵：

| 风险类别 | 识别要点 |
|---------|---------|
| 业务风险 | 竞争加剧、技术路线变更、大客户流失（>30%单一客户）、政策收紧 |
| 财务风险 | 持续亏损无拐点、现金流紧张、收入确认激进 |
| 治理风险 | VIE合规、核心团队稳定性、关联交易、代持未清理 |
| 退出风险 | IPO路径不清晰、行业并购不活跃、回购主体无支付能力 |

**高频遗漏风险自查**：
- 社保公积金是否足额缴纳？（中国PE高频踩雷）
- 知识产权归属公司还是个人？
- 未披露的对外担保/质押？
- VIE架构：外商投资负面清单覆盖否？

### 第四步：筛选结论

| 结论 | 条件 |
|------|------|
| 推荐深入 | 加权均分>3.5，无红线，投资逻辑清晰 |
| 有条件推荐 | 均分3.0-3.5，有可解决的风险点 |
| 暂时搁置 | 均分<3.0但赛道或团队突出，建议跟踪 |
| 建议放弃 | 触发红线/核心逻辑不成立 |

如有基金信息，增加匹配度判断：阶段匹配/行业匹配/规模匹配/DPI要求匹配。

## 输出格式

```markdown
# 项目筛选备忘录

**项目**：深势科技  **行业**：AI for Science / 药物研发SaaS  **阶段**：B轮
**本轮估值**：Pre 25亿 / Post 30亿  **日期**：2026-06-19

## 一句话总结
AI分子模拟平台型公司，自研DeePMD核心算法获国际认可，药企付费客户12家年留存92%，但亏损扩大（年亏1.2亿）且客户集中（前3占65%），估值30亿对应PS 15x偏高需谈判压价。

## 关键信息
| 维度 | 内容 |
|------|------|
| 业务模式 | SaaS+项目制，按分子管线付费（50万-500万/年/管线），聚焦药物分子动力学模拟和材料设计 |
| 核心数据 | 2025A收入2.0亿(+85%)，毛利率72%，经营亏损1.2亿，现金跑道14个月 |
| 团队 | 创始人张XX：普林斯顿应用数学博士+前谷歌AI研究员；CTO李XX：北大化工+MIT博后；团队38人(博士15人)；缺商业化VP |
| 融资历史 | 种子轮: 高瓴 天使轮: 红杉+源码 A轮: 百度风投+CPE B轮(本轮): 启明领投 |

## 投资逻辑评分
| 维度 | 评分 | 要点 |
|------|------|------|
| 赛道 | 4/5 | 全球AI制药CAGR 35%，TAM>500亿；政策明确鼓励；国际竞争有Schrodinger对标 |
| 模式 | 3/5 | 毛利率72%优秀，LTV/CAC约4.5x；但项目制占比40%拖累可复制性，SaaS占比待提升 |
| 壁垒 | 4/5 | DeePMD方法论发Nature+Science共3篇；训练数据集积累5年；客户迁移成本高(需重新训练模型) |
| 团队 | 3/5 | 技术顶尖，学术背书强；缺商业化VP和销售高管；CEO偏技术型，融资/招聘经验一般 |
| 财务 | 2/5 | 增速85%优秀但亏损扩大（-60%净利率）；年度burn 1.5亿 vs 现金2.1亿=14月跑道 |
| 估值 | 2/5 | PS(TTM) 15x vs Schrodinger 9x + 薛定谔AI 11x，溢价30-50%，需压至10x内 |
| **加权均分** | **3.1** | A轮权重：赛道x1.5=6.0 模式x1.5=4.5 壁垒x1.5=6.0 团队=3 财务=2 估值=2 → 23.5/7.5≈3.1 |

## 主要风险（按影响x概率排序）
1. 商业化能力不足 — 影响高 x 概率高：当前12家客户中仅4家续费≥2年，缺成建制销售团队，行业决策链长(6-12月)
2. 人才流失风险 — 影响高 x 概率中：CTO在MIT有tenure track offer；核心算法团队5人无竞业协议
3. 现金跑道压力 — 影响中 x 概率高：14月跑道不足以撑到C轮(行业平均间隔18月)，需B轮+或提前启动C轮
4. 知识产权归属 — 影响高 x 概率低：DeePMD核心算法为开源社区共同开发，虽然3篇论文为团队发表，边界模糊

## 匹配度分析（基金：XX人民币基金三期）
| 维度 | 基金偏好 | 项目 | 匹配 |
|------|---------|------|------|
| 阶段 | A-B轮 | B轮 | ✅ |
| 行业 | 硬科技/企业服务 | AI制药 | ✅ |
| 单笔 | 5000万-1.5亿 | 1亿 | ✅ |
| DPI要求 | 2028年前有退出信号 | 预计2027启动C轮/战投 | ⚠️ 偏紧 |

## 筛选结论
**有条件推荐** — 赛道和壁垒突出(DeppMD技术护城河)，但需满足以下条件：(1)估值压至PS 10x(Pre 18亿以下)；(2)本轮融资用途中≥30%用于组建销售团队；(3)落实CTO留任条件+核心团队竞业。建议下一步：与创始人沟通估值预期和销售团队招聘计划。
```

## 质量标准

- [ ] 每个评分维度有数据或事实支撑（不允许无依据打分）
- [ ] 估值对比选择了合理可比对象（同行业同阶段）
- [ ] 风险清单覆盖业务/财务/治理/退出四类
- [ ] 红线快筛已执行并明确标注结果
- [ ] 阶段差异化权重已应用
- [ ] 所有判断标注了依据和信息局限性

## 关联Skill

- **due-diligence-checklist** — 筛选通过的项目可调用尽调清单生成详细的财务/法律/业务/技术尽调事项
- **investment-memo** — 筛选推荐的项目可调用投决备忘录生成完整的投委会材料
- **return-modeling** — 筛选阶段可调用收益测算快速估算预期 MOIC/IRR
- **term-sheet-review** — 筛选通过后可调用条款审查评估投资条款合理性
- **investment-research-v1/industry-research** — 赛道评估可调用投研行业研究获取市场空间和竞争格局数据

## 失败回退机制

| 步骤 | 失败条件 | 回退目标 | 最大重试 | 不可恢复时升级路径 |
|------|---------|---------|---------|------------------|
| 第零步：红线快筛 | 项目信息不足无法判断红线 | 标注 `[信息不足，建议补充BP]`，跳过红线快筛进入后续流程 | 0 | 提示用户补充项目材料后再执行红线快筛 |
| 第一步：关键信息提取 | BP/CIM 材料缺失或非标格式 | 基于公开信息初步评估，标注 `[信息有限，建议补充BP]` | 1 | 提示用户上传 BP 或 CIM 材料 |
| 第二步：六维量化评分 | 财务/估值数据缺失导致维度无法打分 | 标注 `[数据不足，该维度暂不打分]`，基于已有维度计算加权分 | 1 | 提示用户补充财务数据和估值信息 |
| 第三步：风险矩阵 | 项目信息不足导致风险无法识别 | 输出通用风险框架，标注 `[需补充尽调确认]` | 0 | 提示用户通过 due-diligence-checklist 深入排查 |
| 第四步：筛选结论 | 加权评分无法计算或评分处于边界（3.0-3.5） | 输出"有条件推荐"并标注关键不确定性，建议补充尽调 | 0 | 提示用户补充尽调材料后重新评估 |
